Do you know which CSS styling has highest priority?

Introduction

Many beginners and even intermediate developers have confusion about the priority of the CSS styling. Trust me today after reading this blog you'll be having a solid knowledge on CSS styling priority

Before diving deep let's see what all kind of styling is there in CSS.

  • Type selector
  • Class selector
  • Internal CSS
  • External CSS
  • Id selector
  • Inline style attribute

Type selector

CSS p { color: blue ; font-size:16px; }
h1 { background-color : yellow ; margin: 5px;}

In element type selector , we select the elements by their html tags and apply styling to them as shown in above example.

Type selector has the least priority in CSS styling

Class Selector

HTML <p class="para-class color-blue mb">This is a sample </p> CSS .para-class { color : red; font-size : 16px; }
.color-blue { color: blue; }
.mb { margin-bottom : 10px; }

A html element can have multiple classes and the styling of the last class overrides the pre-existing style if there is a clash.

In the above example the "color-blue" class overrides the font color red of "para-class" and the remaining properties remains unchanged.

Class selector has higher priority than type selector

Internal CSS

HTML <p class="para-class color-blue mb">This is a sample </p> CSS <style>
  .mb{ margin-bottom : 10px; }
  .para-class{ color : red; font-size: 16px; }
</style>

An internal css is written by inserting the style tag inside the head tag of the html file.The styling properties of html elements are written here.

External CSS

HTML <link> rel="stylesheet" href="style.css" </link>

In external css, we code all the styling properties in a seperate external file and link it with the html file by using link tag and including the source of the external css in href attribute

Id selector

HTML <p id="para">This is a sample </p> CSS #para{ color: red; font-size: 16px; }

In id selector, we select the element by their id and apply styling to them either in internal or external CSS.While selecting ID we have to insert hash(#) tag before the ID's name.Html elements should have unique ID . ID selector have second highest priority

Inline style attribute

HTML <p style="color: red">This is a sample </p>

Inline style attributes are those styling where the css properties are written as attirbutes in HTML elements. These styling has the highest priority.

Conclusion

The priority of external and internal css depends on the order which they appear.

If the style tag comes after the link tag then the internal css overrides the styling of external css and vice-versa.

CSS styling priority

Inline CSS >> Id Selector >> Class Selector >> External and Internal CSS >> Type Selector

Finally , we know which kind of CSS styling has what kind of priority.
